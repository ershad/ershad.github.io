Though i can't build huge applications and tools for testing efficiency in both the free and propriety platforms, it's possible to create a small C++ program to perform some-what complex mathematical operations and displaying them in monitor. Such a program has been made and tested it in both GNU/Linux platform and WINDOW$ platform and the result was AMAZING !!. Here it is...

Experiment Desk : P4 2.6GHz Processor, 1.2 GB RAM, ATI on-board Graphics

Testing Tool : A 32 bit C++ program compiled with g++. This program allocates 80000 floats, 10000 X 50 + 1 chars & 4 ints on initial execution. Then it fills 60000  blocks with data ranging from 1 to 49 and generates 10000 names from 5-alphabets combination which will be stored in memory. Next step is to calculate individual sum, individual average and subject wise average of 10000 students with 6 Subjects. Last task is to display all these data as a table on monitor.
